2026 年还在写博客，而且还是从零手写，是不是有点复古？我觉得不。

## 为什么自己搭？

市面上有很多现成的方案：Notion、语雀、Hashnode、Medium... 它们都很方便。但作为一个对「极致性价比」和「自主权」有执念的人，我不想把内容锁在某个平台上。

自己写的博客：

- **数据完全属于自己**：所有文章都是本地 Markdown 文件，Git 管理，随时可以迁移
- **性能由我掌控**：没有广告、没有追踪脚本、没有不必要的 JS，秒开
- **技术栈完全自由**：想加什么功能就加什么，不受平台限制

## 技术选型

这套博客用了几个我很喜欢的工具：

- **[Astro](https://astro.build)**：静态站点框架，默认输出零 JS，性能极致。支持 React/Vue/Svelte 组件，但我选择纯 `.astro` 文件，保持简洁
- **[Tailwind CSS v4](https://tailwindcss.com)**：原子化 CSS，v4 用 CSS 配置代替 JS 配置，更自然
- **[GitHub Pages](https://pages.github.com)**：免费托管，全球 CDN，自动部署
- **[Shiki](https://shiki.style)**：代码高亮，Astro 内置，暗色主题完美

## 后续计划

这个博客会持续记录：

1. **项目展示**：我做的各种项目，从想法到实现
2. **技术笔记**：踩过的坑、学到的东西
3. **工具推荐**：符合「极致性价比」理念的好工具

希望这个博客能陪你走一段路。🚀
