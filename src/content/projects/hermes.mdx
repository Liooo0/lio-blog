## 它是谁

Hermes 是我造的私人 AI 助手，常驻飞书联系人列表里。我喂给它我的日常记录、想法、偏好，它能记住并随时调出来——像一个真正了解我的数字分身。

## 能做什么

- **记忆召回**：问它"我上次聊过什么"→ 从知识库里翻出来。问它"我之前怎么想的"→ 能答
- **跨设备**：飞书 Bot，手机电脑都能聊，随时随地
- **RAG 知识库**：我的个人记忆存向量数据库，不是每次对话重新猜

## 为什么放飞书上

飞书 Bot 不需要装任何 App——只要有飞书就能用。手机、电脑、网页三个端无缝切换。而且飞书的联系人体系让它像一个真正的"联系人"，不是网页上一个冷冰冰的聊天框。

## 技术栈

| 层 | 用什么 |
|---|---|
| 工作流引擎 | Dify（零代码编排） |
| 记忆系统 | RAG + 向量数据库 |
| 对话接口 | 飞书开放平台 Bot API |
| 大模型 | 免费 API（计划升级 DeepSeek 付费版） |

## 为什么做

1. 面试展示：这是一个完整闭环的 AI Agent 项目
2. 自己用：我确实需要一个能记住我所有事情的助手
3. 验证能力：同样的技术栈可以直接给企业做内部知识库问答系统
